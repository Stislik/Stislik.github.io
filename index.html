Produces

Цей виклик API створює наступні типи медіа відповідно до заголовка запиту Accept; тип медіа буде передано в заголовку відповіді Content-Type.
  • application/json

Змінні та їх значення
coord
    lon - Довгота місцезнаходження
    lat - Широта місця розташування
weather
    id - Ідентифікатор погодних умов
    main - Група погодних параметрів (Дощ, Сніг, Хмари тощо)
    description - Погодні умови в групі. Будь ласка, знайдіть більше тут. Ви можете отримати вихідні дані на вашій мові.
    icon -Ідентифікатор піктограми погоди
base -Внутрішній параметр
main
    temp - Температура. Одиниці за замовчуванням: Кельвін, метрична: Цельсій, Імперіал: Фаренгейт
    feels_like - Температура. Цей параметр температури враховує людське сприйняття погоди. Одиниця за замовчуванням: Кельвін, метрична: Цельсій, Імперіал: Фаренгейт
    temp_min - Мінімальна температура на даний момент. Одиниця за замовчуванням: Кельвін, метрична: Цельсій, Імперіал: Фаренгейт
    temp_max - Максимальна температура на даний момент. Одиниця за замовчуванням: Кельвін, метрична: Цельсій, Імперіал: Фаренгейт
    pressure - Атмосферний тиск на рівні моря, гПа
    humidity - Вологість, %.
visibility - Видимість, метр. Максимальне значення видимості - 10 км
wind
    speed - Швидкість вітру. Одиниця за замовчуванням: метрична: метр/сек, метрична: метр/сек, англійська: милі/год
    deg - Напрямок вітру, градуси (метеорологічний)
    gust - Порив вітру. Одиниця за замовчуванням: метрична: метр/сек, метрична: метр/сек, англійська: милі/годину
clouds
    all - Хмарність, %
dt - Час розрахунку даних, unix, UTC
sys
    type - Внутрішній параметр
    id - Внутрішній параметр
    country - Код країни (GB, JP і т.д.)
    sunrise - Час сходу сонця, unix, UTC
    sunset - Час заходу сонця, unix, UTC
timezone - Час у секундах від UTC
id - Ідентифікатор міста. Зверніть увагу, що вбудований функціонал геокодера застарілий.
name - Назва міста. Зверніть увагу, що вбудований функціонал геокодера застарілий.
cod - Внутрішній параметр
