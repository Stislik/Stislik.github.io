Default

Up 
get /weathers
Запит погоди (giveWeather)

Query parameters

q (required)
Query Parameter — Місто, для якого ви хочете отримати інформацію про погоду.
appid (required)
Query Parameter — Ваш особисти OpenWeatherMap API ключ.
units (optional)
Query Parameter — Одиниця виміру температури (метрична, британська або стандартна).
lang (optional)
Query Parameter — Мова для відповіді.

Produces

Цей виклик API створює наступні типи медіа відповідно до Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json
